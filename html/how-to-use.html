How to use

The application check all the Google+ Public stream (posts shared Public or with public Communities) of all GDEs searching for posts that contain the hashtag #gde.

To be processed by the application, other information are needed, so GDEs need to add additional tags to Google+ post they are sharing.


The additional tags are divided in 2 categories:

  • Product Tags

  • Activity Type Tags

Following a cheat sheet of the Tags:


Product Tags
#androidAndroid
#admobAdMob
#adwordsGoogle AdWords
#angularjsAngular JS
#chromeGoogle Chrome
#html5Chrome: HTML5
#dartlangChrome: Dart Language
#cloudplatformGoogle Cloud Platform
#googleanalyticsGoogle Analytics
#googleappsapiGoogle Apps APIs
#googleappscriptGoogle Apps Script
#googledriveCoogle Drive SDK
#glassGoogle Glass
#googlemapsapiGoogle Maps APIs
#googleplusGoogle+ Platform
#youtubeYouTube APIs
#uxdesignUX & Design
Activity Type Tags
#bugreportBugReports: Report Bug on Google APIs
#articleArticle: Write technical article about use cases, tips & tricks, particulare use and experience using an API
#blogpostBlogPost: Maybe less technical than Articles, less formal, but still informative and helpful informations for Developers
#bookBook: A full book about a technology, best practices, examples, guides, etc... all printed on Paper (or digital)
#techdocsTechnical Documentation: Help Improve the documentation on Google Open Source Projects
#translationTranslation: Many articles are in English, but Developers are all around the world and it's helpful if content is translated in their native language
#techtalkTech Talks: Not Scared of the stage? Talk at conventions, GDGs and MeetUps and present technologies to Developers in front of you
#opensourcecodeOpen Source Code: Got a cool library/application and want to share it because can help other developer? Open source it and share with the World
#forumpostForum Posts: Help answering question directly in Forums and StackOverflow
#communityCommunty: Engage with the community live
#videoVideos: Presentations and tutorials presented with a video are more engaging and help convey the passion we feel about the technology
#tutorialTutorials: Lead the developer step by step during his journey for a better knowledge of the technology
